## Key Design Decisions

**Discovery-First Workflow:**
- `search_endpoints` before each data type ensures agents discover available endpoints
- Eliminates hardcoded tool names; adapts to API changes

**Semantic Table Naming:**
- Tables: "ticker_info", "price_history", "options_chain", "financials", "news"
- Enables SQL JOINs and complex cross-table analysis
- Example: `SELECT * FROM options_chain WHERE delta BETWEEN 0.20 AND 0.35`

**Built-in Functions Over Custom Math:**
- Greeks: `bs_delta`, `bs_theta`, `bs_vega`, `bs_gamma`, `bs_rho`
- Technicals: `sma`, `ema`
- Returns: `simple_return`, `cumulative_return`, `sharpe_ratio`

**Conservative Fallbacks:**
- Missing Fear & Greed → stricter IV Rank (≥70 instead of ≥50)
- Missing Trends → focus on news volume as retail proxy
- Missing Earnings data → default to WAIT unless >60 days buffer
- Missing Insider trades → search news headlines for keywords
